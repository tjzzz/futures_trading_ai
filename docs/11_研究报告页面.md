# 研究报告页面设计

> **定位**：整体趋势的宏观把控、预判、专题研究、复盘总结<br>
> **使用场景**：周/月度研究、深度专题、飞书推送管理、预警规则配置

---

## 一、页面定位与价值

### 1.1 与现有页面的区别

| 页面 | 定位 | 使用频率 | 内容深度 |
|:---|:---|:---|:---|
| 总览页 | 实时状态监控 | 高频（每日多次） | 浅层（当前快照） |
| 宏观页 | 四象限深度分析 | 中频（每日1-2次） | 中等（历史趋势） |
| **研究报告页** | 周期性深度研究 | 低频（周/月/专题） | **深度（综合研判）** |

### 1.2 核心价值

1. **周期性报告**：自动生成周报/月报，汇总阶段行情与关键事件
2. **专题研究**：针对特定主题（如"央行购金潮"、"降息周期影响"）的深度分析
3. **复盘总结**：回测预测准确性，优化模型参数
4. **推送管理**：配置飞书定时推送内容、频率、接收人
5. **预警中枢**：统一配置和管理所有监控规则

## 三、功能模块详细设计

### 3.1 周期报告（周报/月报/季报）

#### 3.1.1 自动生成内容

**周报结构：**
```markdown
# 黄金周报 (2026-W21: 05.18-05.24)

## 一、本周行情回顾
- 开盘价: $X,XXX | 收盘价: $X,XXX | 波动区间: $X,XXX-$X,XXX
- 周涨跌幅: X.X% | 振幅: X.X%
- 关键事件: [事件1], [事件2]

## 二、四象限扫描
| 象限 | 信号 | 变化 |
|:---|:---|:---|
| 货币锚 | 利多 | 美元走弱 |
| 流动性 | 中性 | 利差收窄 |
| 风险偏好 | 利空 | VIX回落 |
| 供需 | 中性 | 金银比持平 |

## 三、预测回顾
- 上周预判: 略偏空
- 实际走势: 震荡下跌
- 准确性: ✅ 方向正确

## 四、下周展望
- 综合判断: 中性偏空
- 关键区间: $3300-$3600
- 关注事件: 美联储会议纪要、非农就业

## 五、操作建议
- 短线: 观望为主
- 中线: 关注 3300 支撑
```

#### 3.1.2 生成方式

- **自动**: 每周一 09:00 基于过去7天数据自动生成
- **半自动**: 生成后允许人工编辑关键结论
- **手动**: 支持完全手动创建空白报告

### 3.2 专题研究

#### 3.2.1 专题类型

| 类型 | 触发条件 | 数据源 |
|:---|:---|:---|
| 央行购金专题 | 季度央行净购金数据发布 | WGC + 各国央行公告 |
| 降息周期专题 | 美联储利率决议前后 | CME FedWatch + FRED |
| 地缘风险专题 | VIX 突升 + 新闻事件 | RSS + 事件追踪 |
| 美元周期专题 | DXY 突破关键区间 | FRED + 技术分析 |
| 金银比套利 | 比值突破 80/60 | Yahoo Finance + 计算 |

#### 3.2.2 专题报告结构

```markdown
# 专题：全球央行购金对金价的长期影响

## 一、研究背景
- 现象: 2022-2026 全球央行连续净购金
- 数据: Q1 2026 净购金 290 吨，同比+35%
- 问题: 央行购金对金价形成结构性支撑还是短期脉冲？

## 二、数据分析
- 历史相关性: 央行购金量 vs 金价 (12个月滞后)
- 关键发现: 净购金 > 200吨/季度时，未来12个月金价上涨概率 73%

## 三、机制分析
- 供给端: 央行购金减少市场流通量
- 需求端: 示范效应带动机构配置
- 心理层: 增强黄金"货币属性"认知

## 四、结论与展望
- 短期（1-3月）: 支撑金价底部
- 中期（6-12月）: 与美联储政策博弈
- 长期（2-3年）: 去美元化趋势下的结构性利多
```

### 3.4 预警规则管理

#### 3.4.1 规则类型

1. **阈值预警**: 指标突破设定值
   - 例: 10Y Treasury > 4.5% → 利空黄金

2. **变化率预警**: 短期内大幅波动
   - 例: 黄金单日跌幅 > 2% → 关注支撑

3. **组合预警**: 多指标同时触发
   - 例: VIX > 25 + 美元走弱 → 强利多

4. **时间预警**: 定期提醒
   - 例: 每周一 09:00 发布周报提醒

### 3.5 飞书推送管理

#### 3.5.1 推送类型

| 类型 | 频率 | 内容 | 示例 |
|:---|:---|:---|:---|
| 早班车 | 每日 09:30 | 隔夜变化 + 趋势预判 | "📊 黄金早班车: 隔夜下跌 0.5%，短期看空" |
| 晚报 | 每日 20:00 | 日盘回顾 + 次日预告 | "🌙 黄金晚报: 今日波动 $50，关注明日非农" |
| 周报 | 周一 09:00 | 周行情回顾 + 展望 | "📅 黄金周报: 本周下跌 1.2%，关注美联储" |
| 专题 | 不定期 | 深度研究报告 | "🔍 专题: 央行购金对金价长期影响" |
| 预警 | 实时 | 阈值触发通知 | "⚠️ 预警: 10Y Treasury 突破 4.5%" |

### 4.1 API 设计

```

GET  /api/research/reports           # 报告列表
POST /api/research/reports           # 创建报告
GET  /api/research/reports/{id}      # 报告详情
PUT  /api/research/reports/{id}      # 更新报告

# 专题模板
GET  /api/research/templates         # 专题模板列表
GET  /api/research/templates/{type}  # 特定模板

# 预警规则
GET  /api/research/alerts            # 预警规则列表
POST /api/research/alerts            # 创建规则
PUT  /api/research/alerts/{id}       # 更新规则
DELETE /api/research/alerts/{id}     # 删除规则
GET  /api/research/alerts/history    # 触发历史

# 推送管理
GET  /api/research/pushes            # 推送配置列表
POST /api/research/pushes            # 创建推送
PUT  /api/research/pushes/{id}       # 更新推送
POST /api/research/pushes/{id}/test  # 测试推送
GET  /api/research/pushes/history    # 推送历史

# 复盘数据
GET  /api/research/validation        # 预测验证统计
GET  /api/research/backtest          # 信号回测结果
```
