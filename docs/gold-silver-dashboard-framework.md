## 三、驱动因子矩阵（四象限）

金银价格的四个核心驱动象限，任何时候都有1-2个因子在主导：

| 象限 | 驱动因子 | 核心指标 | 当前状态(2026.5) |
|:----:|----------|---------|:----------------:|
| 🟢 **货币锚** | 美元信用 + 实际利率 | 10Y TIPS、DXY、央行购金量 | 美债破5%→信用松动→利多 |
| 🔵 **宏观流动性** | 美联储政策 + 全球流动性 | Fed fund rate、QE信号、美元流动性 | 降息中但长端飙升→矛盾信号 |
| 🟠 **风险偏好** | 地缘政治 + 美股 | VIX、SPX走势、战争动态 | 美伊+美债→高风险→利多 |
| 🔴 **供需博弈** | 白银挤兑 + 实物需求 | COMEX库存、金银比、矿企成本 | 白银库存紧张→利多 |

## 六、个人操作锚点

### 积存金
- **低吸**：黄金跌至4400附近可考虑小补
- **高抛**：黄金回到5000+且情绪极度乐观时小幅减仓，降成本
- **原则**：几年不动，不在乎短期波动

### 期货
- **补仓**：黄金跌至4200-4400 + 美债收益率见顶信号 → 分批加仓，白银优先
- **减仓**：快速反弹至前高附近 + 量能不足 → 先减部分保灵活性
- **止损**：黄金跌破4000（概率低，但不设不行）

### 通用原则
1. 不恐慌割肉
2. 不盲目追高
3. 拒绝高杠杆
4. 闲钱投资
5. 专注长期逻辑
